## 接口规范及返回

### 1. 返回
- status 状态码
- message 提示信息
- data 数据域

### 2. 状态码与提示信息
#### 2** 成功
```
SUCCESS(200,"success")
```
#### 4** 失败
```
UNKNOWN_ERROR(400,"error") 未知或系统错误
NOT_FOUND(401,"not found") 数据库无记录
RECORD_EXISTS(402,"record exists") 记录已存在
PARAM_EMPTY(403,"param is empty") 参数为空
```

### 4. 数据表设计
#### 1. 文章表
- Integer id; 文章id 主码
- String title; 文章标题 唯一
- String time; 时间 格式 ex 2018-04-16
- String type; 文章类型
- String cover; 封面图片地址
- String content; 文章内容
